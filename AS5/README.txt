To run program:
1. Extract all contents of this zip folder to a unique folder 
2. Ensure a compiler (C) is installed on your computer/ Or connect to the Oregon State University Flip OS1 server
		See those instructions here: https://it.engineering.oregonstate.edu/accessing-engineering-file-space-using-windows-file-sharing
3. Move the new folder with the contents of the extracted zip folder into a codespace with the compiler
4. To compile program run the following command:
		./compileall
